HOW TO USE BLINKTRUST AI
BlinkTrust AI Plugins for IDEs
Follow the steps below showing how to download and install BlinkTrust AI Data Privacy Checker into the IDE of your choice.

  • Select Platform

    To use our plugins for scanning data vulnerability, begin by selecting the IDE of your choice from the list.

  • Choose Plugin

    From the IDE marketplace, search for ‘BlinkTrust AI plugin’ or find the plugin featured on the home screen.

  • Download & Install Plugin

    Click the “Download” button on the BlinkTrust AI details page to install the plugin onto your IDE software.

  • Plugin Installed

    You’re all set! The plugin is available to access from the ‘Plugins’ menu in your IDE software interface.

HOW TO USE BLINKTRUST AI
Finding Vulnerabilities is Easy
Follow the steps below showing how to upload data onto BlinkTrust AI and find inconsistencies as per global complaince standards.

  • Open the BlinkTrust AI Software

    Choose whether to find threats in you data, track your data for vulnerabilities or protect your data from attacks.

  • Choose your Data type

    Select source/platform of your data repository set to be scanned, or upload external file in .csv, .txt and other formats.

  • Upload your Data

    Connect and specify which repository you want to be scanned for vulnerabilities and other security risks.

  • Choose the Security laws

    Browse and select security laws which your data would undergo compliance and privacy standard checks with.

  • Scanning your Data

    You’re all set! Wait for your repositories (or files) to be scanned. A notification will pop-up informing you upon completion.

  • Report Ready!

    Get a detailed report laying out types of sensitive data scanned, top datapoints, and vulnerabilities found in your data.
